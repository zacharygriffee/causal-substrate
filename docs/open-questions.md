# Open Questions

These questions remain open by design. The current code should not be read as having resolved them.

- What exact shape determines branch identity beyond a stable local identifier?
- How should nucleus differ formally from lineage in long-lived continuities?
- What semantics distinguish compatibility, equivalence, and convergence under varying bases?
- How should composite branches compute effective basis?
- How should re-identification uncertainty be scored?
- What rules govern referent split and merge?
- How should salience be represented without collapsing into utility or truth?
- How much inference should the default substrate permit?
- How should volatility be learned, revised, and shared?
- What standard contract should portals expose?
- Which artifact classes should cross local/shared boundaries, and in what form?

Repository additions should preserve room for these questions instead of encoding a premature answer.
